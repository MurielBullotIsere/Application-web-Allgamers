# Allgamers

## Description
Application de communauté sociale conçue pour les joueurs de jeux vidéo.  

Voici ce qui peut être testé :  

- l'inscription d'un nouvel utilisateur,  
- la connexion d'un utilisateur,  
- la gestion des jeux favoris de l'utilisateur,  
- la recherche d'autres joueurs (sections "Joueurs").  

### Accès au jeu 
allgamers.infinityfreeapp.com/Application-web-Allgamers/index.php

Installer xampp

Y cloner ce dépot github
